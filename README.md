Prerequisites 
  - Node JS
  - Handlebars
  - Sass -> SCSS style
  - Gulp

## Run project
Install nodemon globally using the following command

```
npm i -g nodemon
```

```
npm install
```

```
gulp watch &
```

```
nodemon app.js
```
